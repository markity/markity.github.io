参数说明

参数 说明
-c 表明进行压缩操作
-z 表明使用gz格式对文件压缩
-v 在压缩过程中打印压缩项
-f 指定目标文件名. 其参数紧跟其后, 因此f参数必须在所有参数之后
-x 表明进行解压缩操作

移动( mv )

移动文件或文件夹

注意: 若新的路径已有同名文件, 则默认覆盖.若希望在遇到同名文件时询问, 可以使用参数-i, 命令变为> mv -i 文件或文件夹路径 新的路径.值得一提的是此命令可用作重命名文件或目录

Docker Ubuntu解决中文乱码

Posted on 2019-08-20

问题引出

Docker中的Ubuntu镜像特别干净, 不但没有vi, wget等基础工具, 甚至连字符集也没几个, 导致无法输入中文

查看镜像内的字符集, 发现仅仅自带3个, 默认使用的字符集是POSIX

设置字符集

选择字符集

要使用中文, 最合适的字符集莫过于utf8了, 它支持了全世界大多数文字字符, 俗称”万国码”

我们可选的utf8字符集有zh_CN.uft8或者en_US.utf8, 实际上它们都是utf8字符集, 对字符的支持等价。唯一差别只是标识了用户使用的语言而已, 系统的菜单、程序的工具栏语言、输入法默认语言随它而定

服务器建议使用en_US.utf8, 避免踩坑
